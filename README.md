# Movie Booking Backend

Backend system for an online movie ticket booking platform built with Spring Boot.

---

## Features

- JWT Authentication
- Role-based Authorization
- Movie Management
- Cinema & Showtime Management
- Seat Booking
- Redis Seat Locking
- Booking History
- Online Payment Integration
- Swagger API Documentation
- Docker Support

---

## Tech Stack

### Backend
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- Maven

### Database
- MySQL
- Redis

### Tools
- Docker
- GitHub Actions
- Swagger
